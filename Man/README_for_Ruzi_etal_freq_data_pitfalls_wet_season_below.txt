Data used in:
Can variation in seed removal patterns of Neotropical pioneer tree species be explained by local ant community composition?
Selina A. Ruzi, Paul-Camilo Zalamea, Daniel P. Roche, Rafael Achury, James W. Dalling, and Andrew V. Suarez.
Published in Biotropica
The freq data pitfalls wet season below file is intended to document the dataset analyzed and to facilitate reanalysis. It contains frequency data of ant species captured at different sites collected using pitfall or subterranean traps placed in the wet season. Data was collected from five sites at Barro Colorado Island, Panama. This file is used to compare ant communities captured by pitfall traps to the subterranean traps.

2. Number of cases/rows: 41
