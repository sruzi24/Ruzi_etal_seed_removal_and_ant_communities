Data used in:
Can variation in seed removal patterns of Neotropical pioneer tree species be explained by local ant community composition?
Selina A. Ruzi, Paul-Camilo Zalamea, Daniel P. Roche, Rafael Achury, James W. Dalling, and Andrew V. Suarez.
Published in Biotropica
The raw data belowground file is intended to document the dataset analyzed and to facilitate reanalysis. It contains raw incidence data of ant species captured in different samples collected from subterranean traps placed at different sites at Barro Colorado Island, Panama and associated either with seed caches from Neotropical pioneer tree species or control caches or passive traps. This file is used to generate belowground species accumulation curves, belowground sampling coverage, and species estimates between traps associated with control caches (bead caches and passive samples) and traps associated with seed caches.

NA = data not available

1. Number of variables: 113
2. Number of cases/rows: 160
3. Variable List: 
Sample: The unique collection code associated with the trap. If the trap had no specimens in it during specimen process then it was given empty sample.
Plot: One of five pre-established sites on Barro Colorado Island, Panama where sampling took place.
Experiment: Defaults to "below" for this dataset as all traps were subterranean traps.
Replicate: Which side of the pre-established plat traps were placed if known.
Year: Sample collection year, YYYY. All was \code{2013} for this dataset.
Season: Season in which the samples were collected. All in the "wet" season for this dataset.
Position: Position the trap was placed with respect to the buried seed or bead cache if known.
Species: Neotropical pioneer tree species. "OcPy" = Ochroma pyramidale, "ZaEk" = Zanthoxylum ekmanii, "CeLo" = Cecropia longipes, "glass" = silica bead control, "TrBl" = Trema micrantha black seed morph, "ApMe" = Apeiba membranaceae, "JaCo" = Jacaranda copaia, "passive" = passive control
Sample_num: The number of samples this row represents.
Data_type: The type of data that is represented here. Defaults to "binary" as this is incidence data.
